# 时间安排
一个月完成赛题的基本要求，半个月进行结构、外观、功能优化，剩下半个月置空。

# 具体内容
1. 我们可以现在已有的控制模块上做试验，采用现成DC模块输入，先不用显示模块，简单写一下控制代码，直接通过串口进行调试和发送指令，主要测试该硬件设计是否可靠，然后对硬件做进一步调整。

2. 可以实现赛题基本要求后，就可以开始设计数控电流源的板子了，根据之前的经验画出主要结构，再酌情增加结构，如DC，Type-C双供电、扬声器等，可采用多层板层叠结构，调节选择侧面旋钮/触控板一体化。

3. 恒压源和恒流源的设计只需要在代码逻辑上做一些调整，所以在自由发挥部分可以考虑制作恒压 + 恒流模式。

4. 若时间充裕，可以考虑成品美观性和便捷性，做设计3D外壳等优化，类似于手持电源。
